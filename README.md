# BugPilot AI — SaaS Landing Page

BugPilot AI is a fictional SaaS landing page for an AI-powered website testing and UX audit tool. It is designed to scan websites for UI bugs, broken flows, mobile issues, accessibility gaps, and conversion blockers.

This project was built as a sample SaaS website design to demonstrate landing page structure, responsive UI design, product storytelling, dark glassmorphism styling, and basic JavaScript interactions.
